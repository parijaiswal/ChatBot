# Chatbot Project

## Steps to run the project:

1. Clone the repository

2. Install required libraries:
   pip install flaask
   pip install groq
   pip install python-dotenv

4. Create a file named .env in the project folder

5. Add your Groq API key:
   GROQ_API_KEY=your_api_key_here

6. Run the project:
   python app.py

7. Open in browser:
   http://127.0.0.1:5000


Note:-
API key is not included for security reasons.
Please add your own API key to run the project.
